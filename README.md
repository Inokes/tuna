# tuna - binary package manager that just works

Most package managers today are complicated, don't comply with eachother and are like siblings. They come from the same place (they use binaries) but they all use different configurations, setups and all sorts of gimmicks.

Tuna solves this by using plain binaries, no configuration except when it's explicitly stated and installs them directly to your binary folder.

# Why the name "tuna"?

When i first thought of this program, i wanted to name it spm (Simple Package Manager). I instantly thought of naming it SPAM, but it would sound shady. So i thought of something close to SPAM and named it tuna. That's the history.
